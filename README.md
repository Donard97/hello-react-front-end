## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

- NodeJS - [v16.x](https://nodejs.org/en/)
- NPM

### Setup

- git clone git@github.com:Donard97/hello-react-front-end.git
- cd hello-rails-front-end

### Install

- yarn

### Usage

- npm run start
